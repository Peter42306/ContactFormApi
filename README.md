# ContactFormApi

REST API for handling Contact Form submissions from static websites and web applications with SendGrid integration.

## Features

- ASP.NET Core 8
- Layered Architecture
- REST API
- Support for multiple applications
- SendGrid integration *(in progress)*
- PostgreSQL *(in progress)*

## Project Structure

- ContactFormApi.Api
- ContactFormApi.Application
- ContactFormApi.Domain
- ContactFormApi.Infrastructure

## Status

Under development.
